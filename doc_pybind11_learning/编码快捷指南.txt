其他这些 snippet 也是一样的用法
比如：

isisbrief → 插入简短 Doxygen 注释
isisfunc → 插入函数注释模板
isisclass → 插入类注释模板
binddoc → 插入 pybind 绑定说明
gxmeta → 插入作者/日期块
gxhistory → 插入历史记录项
gxnow → 插入当前时间戳
为什么有时候看不到 snippet
常见原因有几个：

你当前不是 C/C++ 文件
VS Code 还没重新加载 snippet
自动补全没有弹出来
Tab 没绑定为插入 snippet
如果刚改完 isis.code-snippets，可以试试：

保存文件
重新打开一个 .cpp 文件
输入 gxnow
按 Ctrl+Space
还不行的话，重载一下 VS Code 窗口通常就好了。

小提醒
你现在打开的是定义文件 isis.code-snippets。
这个文件的作用是“注册模板”，真正使用要去别的代码文件里输入前缀，比如 gxnow。
